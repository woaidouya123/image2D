开始 文档 关于 帮助 RFC

Guider

开发这个库的初衷是希望越来越多的人可以更自由的发挥自己的创意，感受绘图的乐趣，并把积累的经验分享出来，推动库本身的发展，从而不断优化！因此，这个库永远都是自由、开源、讨论和进步的。

我们欢迎任何人的加入，具体的内容你可以通过访问贡献指南来找到你可以参与的部分。当然，任何时候，特别是遇到疑惑，提issue都是不错的选择！

Info

目前项目处于RFC阶段，对未来版本有任何改进意见的，都可以在RFC issue中讨论交流，如果你想使用image2D绘制一些图形和别人分享（比如RFC中希望引用例子），你可以在Image Demo中参与用例开发，用例的在线访问地址是Image Demo Inline。

另外，本项目加入了开源中国并在其开源，欢迎大家关注。
